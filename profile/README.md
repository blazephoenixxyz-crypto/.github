### What we build

- **On-chain quotes.** Every swap quote is computed on-chain by the Quoter contract — no off-chain price feed to trust. The API and site are thin mirrors of the chain.
- **Surplus to the user.** When execution beats the quote, the difference is returned — fee-exempt, in the contract, not in a promise.
- **Provably-solvent staking.** `isSolvent()` is public on-chain; anyone can verify at any time.
- **Free, open developer surface.** Public quote API (no key, open CORS), a TypeScript SDK, and a zero-custody Telegram bot example. Built for the agent economy: `llms.txt`, OpenAPI, self-describing discovery.
